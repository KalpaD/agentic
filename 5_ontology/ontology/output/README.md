# output/

Generated files. Everything here is safe to delete and regenerate.

## graph.html

An interactive visualisation of the cognee knowledge graph, generated by `explore/inspect_graph.py`.

```bash
python explore/inspect_graph.py
open output/graph.html
```

**How it was generated:**
cognee's `visualize_graph()` function reads the Ladybug graph database, preprocesses the nodes and edges, and injects them as JSON into a self-contained HTML template that includes a force-directed graph renderer. No internet connection is required to open it.

**What you can do in the browser:**
- Drag nodes to rearrange the layout
- Hover over nodes to see their type and attributes
- Hover over edges to see the relationship label
- The layout separates node types visually (Entity, EntityType, DocumentChunk, etc.)
